LRST Interactive Dynamics Explorer

An interactive visualization tool for Loss-Recovery Systems Theory (LRST).

This tool demonstrates how small asymmetries in loss pathways can produce large-scale divergence in system behavior over time.

🔬 What this shows

The model is based on a simple iterative equation:

Xₙ₊₁ = Xₙ + G − L_sym(1 + ε(Xₙ))

Where:

X = system state (normalized)

G = input drive

L_sym = baseline loss

ε(X) = state-dependent asymmetry

⚡ Key behaviors demonstrated

Threshold dynamics
Crossing a critical value changes whether the system grows or decays

Bifurcation
Nearly identical starting conditions diverge dramatically over time

Hysteresis (memory)
System behavior depends on history, not just current state

Asymmetric loss dominance
Retention pathways (ε₋) have stronger impact than dissipative ones (ε₊)
